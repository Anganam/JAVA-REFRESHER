# Module 6 Using Loop Constructs


## Lesson 1: Create and use while loops



## 1.1 Creating and using while loops
Basic form of a `while loop`
````
while(<boolean-tes>)
    statement
````
Use curly braces  
````
while(<boolean-tes>){
    statement
}
````

Example
````
int x = 10;
while(x > 0){

    System.out.println("x is ");
    x--;
}
    statement
````
